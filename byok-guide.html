怎么接入 AI 服务（2.0 版）

腕语用你自己的 API key 直连 AI（BYOK），不经任何中间服务器。2.0 起接入只需三步：选服务 → 贴 key → 同步手表。key 只存在你 iPhone 和手表的钥匙串里。

本页内容 ① 三步接入（App 内操作） ② 各家服务去哪拿 key ③ 模型三槽位：对话 / 视觉 / 语音转录 ④ 本地模型（Ollama / LM Studio） ⑤ 常见问题

① 三步接入

  1. 选服务：打开 iPhone 端腕语 →「AI 服务」→「添加 AI 服务」（首次启动会自动弹出设置向导，一样的流程）。从预设列表里挑一家——国内推荐硅基流动（直连、模型全、按量便宜），也有通义千问、DeepSeek、智谱、Kimi、Claude、OpenAI、Gemini 等。
  2. 贴 key：进入这家服务的页面，把 API key 粘贴进去、保存——App 会自动打一个零费用请求验证，显示「已就绪」即成。端点和常用模型都已预填好，不需要再填任何 URL 或模型 ID。
  3. 同步手表：戴上手表打开腕语，回 iPhone 点「立即同步到手表」。完成，抬腕即问。

可以同时接入多家服务，互不冲突；每家页面里勾选的模型会出现在手表的模型切换列表里。

③ 模型三槽位

  1. 对话模型：日常聊天用，手动选。
  2. 视觉模型：带图片提问时自动切换。默认「自动」——对话模型自己会看图就直接用它，不会就用同一家服务里的视觉模型。
  3. 语音转录模型：自研语音输入的转写。默认「自动」——用对话模型所在服务的转写模型（如硅基流动的 SenseVoice）。

「自动」会跟着你换对话模型联动变化；你手动选过就固定，不会被系统悄悄改。

④ 本地模型（完全离线私有）

  1. 电脑上跑 Ollama 或 LM Studio。
  2. App「添加 AI 服务」选对应预设，把端点里的 IP 改成那台电脑的局域网地址（key 随便填一个占位）。
  3. iPhone / 手表和电脑连同一个 WiFi 即可使用，数据不出内网。

⑤ 常见问题

  1. 保存 key 后显示「不可用 / HTTP 403」：403 常见于当前网络地区被该服务屏蔽（如国内直连 Anthropic/OpenAI 官方端点）——换国内直连的服务，或在服务页面把端点改成可用的中转地址。
  2. 手表提示「未配 key」：手表打开腕语，iPhone 点一次「立即同步到手表」。
  3. 模型列表太长：在服务页面把不常用的模型取消勾选，手表列表会跟着变短。
