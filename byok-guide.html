怎么获取并填入 API key

WatchChatBot 用你自己的 API key 直连 AI(BYOK），不经我们的服务器。任选一家，几分钟搞定。key 只存在你手机和手表的钥匙串里。

选你要用的服务 ① 通义千问(国内首选，直连） ② Claude（Anthropic） ③ 其他（OpenAI 兼容：硅基流动 / DeepSeek / OpenRouter…）

① 通义千问（DashScope）

人在中国大陆首选：直连、不用代理、按量便宜。

  1. 打开阿里云百炼控制台 bailian.console.aliyun.com，登录并开通「模型服务」。
  2. 在「API-KEY」页新建一个密钥，复制（形如 sk-…）。
  3. 回 App「API Keys」区，在「通义千问」那行粘贴、保存 —— 会自动校验，显示「已就绪」即可。
  4. 手表打开 WatchChatBot，回 iPhone 点「立即同步到手表」一次。

② Claude（Anthropic）

中国大陆直连常被拦（HTTP 403）。若报 403：用代理，或在「高级 → 自定义端点」填一个 Anthropic 兼容中转端点。

  1. 打开 console.anthropic.com 注册、充值（Claude 需预充额度）。
  2. 在「API Keys」创建密钥，复制（形如 sk-ant-…）。
  3. 回 App「API Keys」区，在「Anthropic (Claude)」那行粘贴、保存。
  4. 手表打开 App，iPhone 点「立即同步到手表」。

③ 其他（OpenAI 兼容，通用）

硅基流动、DeepSeek、OpenRouter 等任何 OpenAI 兼容平台都能接。硅基流动国内直连、便宜，推荐新手。

  1. 注册任一 OpenAI 兼容平台（如硅基流动 siliconflow.cn），拿到 API key。
  2. App「API Keys」：在最接近的供应商行粘贴 key（用 OpenAI 通道就填 OpenAI 行）。
  3. 「高级 → 自定义端点」：填该平台完整的 chat 端点 URL（如 https://api.siliconflow.cn/v1/chat/completions）。
  4. 「默认模型 → 自定义模型」：把平台模型页的模型 ID 原样复制进去，设为默认。
  5. 保存后重新「测试」key，再同步到手表。
隐私：API key 只存于你手机与手表的 Keychain，不经任何第三方服务器。 · WatchChatBot
